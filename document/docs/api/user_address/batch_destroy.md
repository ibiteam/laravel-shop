# 批量删除地址

* 删除多个选中的地址

### 请求URL

* /api/v1/address/batch_destroy

### 请求方式
* POST

### 请求参数

| 字段  | 是否必填 | 字段类型  | 说明                |
|:----|:-----|:------|:------------------|
| ids | Y    | array | 地址id数组 ids =【1,2】 |



### 返回示例

```
{
    "code": 200,
    "message": "保存成功",
    "data": null
}
```
