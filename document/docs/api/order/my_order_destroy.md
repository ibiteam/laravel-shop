# 删除订单

* 当前用户删除已完成或已取消的订单

### 请求URL

* /api/v1/order/my/destroy

### 请求参数

| 字段              | 字段类型    | 是否必须 | 说明       |
|-----------------|---------|------|----------|
| no              | String  | Y    | 订单编号     |

### 请求方式
* POST

### 返回示例

```json
{
    "code": 200,
    "message": "删除订单成功",
    "data": null
}
```
