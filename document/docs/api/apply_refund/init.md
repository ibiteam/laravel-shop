# 初始化申请售后

* 获取申请售后类型与售后商品信息

### 请求URL

* /api/v1/order/apply_refund/init

### 请求方式

* GET

### 请求参数

| 字段              | 字段类型    | 是否必须 | 说明     |
|-----------------|---------|------|--------|
| order_no        | string  | Y    | 订单编号   |
| order_detail_id | integer | Y    | 订单明细ID |
